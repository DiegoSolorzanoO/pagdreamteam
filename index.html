Aprendizajes

Hyper Text Markup Language tambien conocido como HTML fue el primer lenguaje de programación orientado al manejo de paginas web para el usuario, incluyendo solo algunas pocas herramientas para el manejo de cuadros de textos y orientaciones, convirtiéndose en la base de programación de paginas web. Con el paso del tiempo HTML desarrollo diferentes versiones del mismo lenguaje implementando nuevas herramientas para hacer posible la implementación de: buscadores, tiendas en línea y banca electrónica.

El lenguaje de programación Cascading Style Sheets tambien conocido como CSS es un lenguaje de programación que funciona como soporte para el diseño del desarrollo de páginas web por medio de HTML, este lenguaje de programación describe la manera en la cual los elementos de las páginas web son desplegados, así como los estilos que llevan los mismos elementos, es decir que CSS funciona como un lenguaje que sirve para crear un ambiente más agradable para el usuario.

JavaScript es un lenguaje de programación que puede ser utilizado desde cambiar elementos de HTML hasta definir el comportamiento de la página. Emplea el uso de librerías para no repetir ciertos códigos. Otro atributo que tiene es Content Delivery Network o CDN, cuya idea principal es para utilizar todo el contenido de una página (imagenes, animaciones) en múltiples servidores en vez de solamente uno local, lo que de algún modo agiliza el proceso de carga.
